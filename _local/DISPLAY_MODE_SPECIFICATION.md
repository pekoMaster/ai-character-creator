## 📋 顯示模式說明

### CSS Class 定義

```css
/* 完整版專屬 (精簡版隱藏) */
.full-mode-only {
  display: none; /* 預設隱藏 */
}
.full-mode .full-mode-only {
  display: flex; /* 完整版顯示 */
}

/* 精簡版專屬 (完整版隱藏) */
.simple-mode-only {
  display: flex; /* 預設顯示 */
}
.full-mode .simple-mode-only {
  display: none; /* 完整版隱藏 */
}
```

### 使用原則

- **無 class**: 兩個模式都顯示
- **`full-mode-only`**: 僅完整版顯示 (精簡版隱藏)
- **`simple-mode-only`**: 僅精簡版顯示 (完整版隱藏) - **應避免使用**

## 🎯 設計理念

### 精簡版 (Simple Mode)
- **目標**: 快速創建角色卡的核心資訊
- **適用**: 新手用戶、快速原型、簡單角色
- **欄位**: 只包含必要的基本資訊

### 完整版 (Full Mode)
- **目標**: 創建詳細、完整的角色卡
- **適用**: 進階用戶、複雜角色、TRPG 專業需求
- **欄位**: 包含所有可用欄位和進階功能
