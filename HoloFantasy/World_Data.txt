# Module: Holo Fantasy Static Map (v1.0)
> **Parent Kernel**: Holo Fantasy Narrative RPG (v3.0)
> **Type**: Static Geography / Immutable Navigation
> **Purpose**: Provide absolute coordinates and travel logic to prevent narrative hallucinations.

## 1. [GEOGRAPHY_LOCK] 地理錨點協議 (Immutable Map)
> **[System Directive]**: 本地圖的「方位」、「氣候」與「相鄰關係」為絕對固定 (Immutable)。
> **[Navigation Rule]**: 當玩家移動時，必須遵守地理邏輯（例如：從北方雪原去南方海島，必須經過中央王都或搭船繞行，不可能直接走路抵達）。

### 【中央區域：世界的心臟】
**[The Central Capital / 聖索王都 (Seiso Capital)]**
*   **位置**: 大陸正中央。
*   **地貌**: 平原、河流交匯處、巨大的白色城牆。
*   **氛圍**: 秩序、繁榮、宗教中心。擁有巨大的「大聖堂」與「冒險者總公會」。
*   **常見居民**: 人類、神職人員、騎士。(Sora, Nodoka, A-chan, Subaru)

**[The Wilds / 迷途森林] (新手區)**
*   **位置**: 環繞王都外圍的緩衝地帶。
*   **地貌**: 茂密的綠色森林、古代遺跡殘骸、史萊姆與野狼出沒。
*   **氛圍**: 充滿未知但相對安全，冒險者的起點。
*   **常見居民**: 遊俠、德魯伊、半獸人。(Fubuki, Mio, Watame, Fauna)

### 【北方：凍土與戰爭】
**[Frostgard / 霜守領]**
*   **位置**: 大陸正北方。
*   **地貌**: 永凍冰原、針葉林、暴風雪山脈。
*   **氛圍**: 嚴酷生存、傭兵文化、軍事化管理。擁有「白獅要塞」。
*   **常見居民**: 雪精靈、獸人傭兵、流亡者。(Lamy, Botan, Koyori)

### 【西方：鋼鐵與荒漠】
**[Iron Wastes / 鋼鐵荒原]**
*   **位置**: 大陸正西方。
*   **地貌**: 紅土荒漠、巨大的工廠煙囪、廢鐵山、充滿油污的河流。
*   **氛圍**: 蒸汽龐克、混亂、無政府狀態、瘋狂實驗。
*   **常見居民**: 工匠、地精、機械族、瘋狂科學家。(Pekora, Roboco, Haachama, Kaela)

### 【東方：和風與神秘】
**[Hinoyama / 日之山]**
*   **位置**: 大陸正東方，隔著迷霧山脈。
*   **地貌**: 櫻花林、竹林、神社、浮空島。
*   **氛圍**: 東方奇幻、妖怪橫行、神秘結界。
*   **常見居民**: 鬼族、武士、陰陽師、忍者。(Miko, Ayame, Iroha, Fubuki-白上形態)

### 【南方：海洋與財寶】
**[Atlantis Archipelago / 亞特蘭提斯群島]**
*   **位置**: 大陸正南方，沿海與群島。
*   **地貌**: 熱帶沙灘、珊瑚礁、海底都市遺跡、海盜港口。
*   **氛圍**: 自由、貿易、海怪威脅、寶藏傳說。
*   **常見居民**: 海洋種族、海盜、商人。(Marine, Gura, Aqua, Chloe)

### 【特殊區域：異界與禁地】
**[The Abyss / 深淵魔境]**
*   **位置**: 地圖的邊緣（可能是極西的裂縫，或地底深處）。
*   **地貌**: 紫色天空、扭曲的重力、黑色尖塔。
*   **氛圍**: 絕對的惡意、高濃度魔力、魔王軍大本營。
*   **常見居民**: 惡魔、死靈、魔王軍幹部。(Laplus, Towa, Choco, Shion)

**[The Library / 虛空圖書館]**
*   **位置**: 存在於空間夾縫中，入口隨機出現（或在特定遺跡）。
*   **地貌**: 無限延伸的書架、靜止的時間。
*   **常見居民**: 時間/文明守護者、禁忌記錄者。(Kronii, Mumei, Shiori, Ina)

## 2. [TRAVEL_LOGIC] 旅行邏輯
> **[AI Directive]**: 玩家在區域間移動時，必須消耗時間 (Days) 並觸發旅途事件。

*   **鄰近區域移動** (例: 王都 -> 森林): 耗時 **1 天**。低風險。
*   **跨區域移動** (例: 森林 -> 北方霜守領): 耗時 **3~7 天**。需進行 [生存檢定] (消耗糧食/露營)。
*   **長途跋涉** (例: 北方 -> 南方): 耗時 **10~20 天**。極高機率觸發 [區域衝突事件] 或 [Boss 遭遇]。

## 3. [NAV_AID] 導航輔助 (Designer's Note)
*   當玩家說「我要往北走」時 -> 氣候變冷，遭遇 Lamy/Botan 類型 NPC。
*   當玩家說「我要往西走」 -> 看到工廠煙囪，遭遇 Pekora/Roboco 類型 NPC。
*   **絕對禁止** 瞬間移動或地理跳躍 (除非使用高階傳送魔法)。
