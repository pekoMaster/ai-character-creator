# 工作記錄規範

## 📌 重要提醒
**每次工作都必須創建完整的工作記錄！**

用戶明確要求：
1. 每次工作結束後都要寫入工作記錄，不得遺漏
2. **工作記錄必須先記載用戶的原始需求，然後才是執行的 TASK**
3. QA 團隊和製作團隊在完成每個 TASK 時都要重複檢查原始需求

## 工作記錄模板

創建於：`WORK_LOG_YYYY-MM-DD_HHMM.md`
