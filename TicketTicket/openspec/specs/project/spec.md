# Zaiko Ticket Companion Matching Platform - 專案規格

## 概述

### 專案名稱
Zaiko 票券同行配對平台 (TicketTicket)

### 目標
建立一個行動優先的網頁應用程式，讓持有 Zaiko 電子票券的用戶可以找到同行夥伴一起入場。核心概念是「共同入場」(Shared Entry)，而非票券轉售。

### 核心原則
- **不是票券轉售**：平台用於配對同行者，不涉及票券買賣
- **無線上支付**：所有費用分攤在線下面對面進行
- **分攤成本模型**：要價不得超過票價除以人數

## 用戶角色

### 主辦方 (Host)
- 持有主票券的用戶
- 可以發布同行者徵求
- 管理配對請求
- 與配對者溝通

### 賓客 (Guest)
- 尋找同行機會的用戶
- 瀏覽可用票券
- 申請加入
- 與主辦方溝通

## 功能範圍

### MVP 包含
1. 票券刊登系統
2. 搜尋與探索
3. 配對流程
4. 私訊聊天
5. 評價系統
6. 免責聲明

### MVP 不包含
- 真實用戶驗證
- 線上支付
- 推播通知
- 後台管理
